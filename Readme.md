# KeepIt - Minimalistic Notes App

KeepIt is a lightweight and minimal note-taking application designed for speed, simplicity, and clarity.

Built with a focus on performance and clean user experience, KeepIt helps you capture ideas quickly without distractions.

---

## Features

- Fast and responsive performance
- Simple and intuitive user interface
- Easy note creation and management
- Reliable saving system
- Clean and minimal design
- Privacy-first app

---

## Project Goals

KeepIt aims to:

- Deliver a smooth and lag-free experience
- Maintain a distraction-free UI
- Stay lightweight and efficient
- Continuously improve usability
